# How to lay things our on your page

## The easiest method: a grid system

Bootstrap already gives us a grid system! It lets us lay out things in rows and columns in a pretty elegant way.

Bootstrap's grid system: <http://getbootstrap.com/css/#grid>

## When the grid system doesn't work:

We can't use the grid system when we have elements that don't fit nicely into it; sometimes we've got to customize our layout a bit more. Things that Bootstrap's grid isn't as good at:

- Laying out elements that don't fit nicely into the widths of the grid system
- Perfectly centering elements (horizontally **and** vertically)
- Navbar and navigation elements
- Horizontal lists of elements
- 5-column grids

# `display`

Enter the `display` style!

We can throw the `display` style on an element to change how it's laid out on the page.

There's lots of values that we can assign to the `display` style, but we'll mainly only see a couple:

- `display: block`
- `display: inline`
- `display: none`
- `display: inline-block`
- `display: flex`

> _Read the docs:_

> - <http://www.w3schools.com/cssref/pr_class_display.asp>
> - <https://developer.mozilla.org/en-US/docs/Web/CSS/display>

--------------------------------------------------------------------------------

# How the `display` style changes elements

property                                                                               | none | block | inline | inline-block | flex
-------------------------------------------------------------------------------------- | ---- | ----- | ------ | ------------ | ----
actually displays on the page                                                          | no   | yes   | yes    | yes          | yes
by default, expands to fill the width of the parent container (stacks vertically)      | no   | yes   | no     | no           | yes
by default, constrains to the content of the children containers (stacks horizontally) | no   | no    | yes    | yes          | no
can set width and height                                                               | no   | yes   | no     | yes          | yes

# Flexbox

Flexbox is complicated, but powerful! For everything that you can't handle with the Bootstrap Grid System, I'd recommend using flexbox.

The easiest way to learn flexbox: <http://flexboxfroggy.com/>

The absolute best guide to flexbox: <https://css-tricks.com/snippets/css/a-guide-to-flexbox/>

## How to center yo shit with flexbox (horizontally **and** vertically!)

```
<head>
    <style>
        .my-centering-container {
            height: 300px; /* or whatever */
            width: 300px; /* or whatever */
            background-color: MediumOrchid; /* or RebeccaPurple */
            display: flex;
        }
        .my-centered-element {
            margin: auto;
        }
    </style>
</head>
<body>
    <div class="my-centering-container">
        <div class="my-centered-element">
            <img src="https://media.giphy.com/media/OcZp0maz6ALok/giphy.gif" />
        </div>
    </div>
</body>
```

# Media queries

**Media queries** let us change what styles are applied based on the width of the page (or other things).

This means that we can apply a completely different set of styles based on if a user is on a desktop, tablet (landscape), tablet (portrait), phablet, phone, or Nokia 3310!

![](http://i.huffpost.com/gen/3371794/images/o-NOKIA-3310-facebook.jpg)

Bootstrap uses media queries to change the width of columns based on how wide the screen is.

## A simple example:

```
<head>
    <style>
        p {
            background-color: white;
        }

        @media (max-width: 600px) {
            p {
                background-color: gold;
            }
        }

        @media (max-width: 300px) {
            p {
                background-color: forestgreen;
            }
        }
    </style>
</head>

<body>
    <p>
        Feel da rhythm Feel da rhyme Get on up It's bobsled time
    </p>
</body>
```

## An advanced example:

```
<head>
    <style>
        .bobsled-time {
            display: flex;
            flex-direction: row;
        }

        @media (max-width: 400px) {
            .bobsled-time {
                flex-direction: column;
            }
        }
    </style>
</head>

<body>
    <div class="bobsled-time">
        <p>
            One for da money
        </p>
        <p>
            Two for da show
        </p>
        <p>
            Three for da honeys
        </p>
        <p>
            Go go go
        </p>
    </div>
</body>
```
